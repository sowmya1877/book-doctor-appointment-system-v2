# 📋 Requirement Analysis

## Functional Requirements

- User Registration
- User Login
- Doctor Login
- Admin Login
- Search Doctors
- Book Appointment
- Cancel Appointment
- View Appointment History
- Manage Doctor Details
- Manage Patients

## Non-Functional Requirements

- Responsive Design
- Secure Authentication
- Fast Performance
- Easy Navigation
- Data Privacy

## Software Requirements

- React.js
- Node.js
- Express.js
- MongoDB
- Visual Studio Code
- Git
- GitHub

## Hardware Requirements

- Minimum 4 GB RAM
- Intel i3 or above
- Internet Connection
